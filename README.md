Project Contains Hibernate Native Exercise . To **run** project follow the steps. 

## Create and Run jar Executable File By Following Command
```sh
mvn clean compile assembly:single
```
 This Command clean the Dependencies of this project if present. and create **target** folder with executable **jar** file.
 ```sh
 java -jar target/JavaPersistentHiibernate-1.0-SNAPSHOT-jar-with-dependencies.jar
 ```
